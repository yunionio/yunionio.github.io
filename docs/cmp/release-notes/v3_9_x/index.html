  • 【API网关】通过apigateway类型的endpoint上传镜像报错
  • 【调度】采用共享存储的主机迁移时不应该检查目标宿主机的存储容量限制
  • 【裸金属】未使用网卡缺少index或nic_type，导致宿主机转换回裸金属失败
  • 【i18n】修正重复添加删除i18n信息的错误
  • 【多云纳管】防止Cloudpods嵌套纳管自己
  • 【镜像】arm镜像上传后显示CPU架构为x86

  • 【主机】支持通过快照创建磁盘
