Common System Images

Display download links for common images.

Community Images​

Starting from 3.11, the platform has added "Community Images" functionality in the frontend system image console, integrating common publicly downloadable operating system images. Users can directly click to import. Integrated image information is maintained here. If you find download links are invalid, or wish to add new images, you can submit a PR to this document.

Public Images​

If "Community Images" cannot find the image download you need, please refer to the following image download addresses provided by common operating systems:

CentOS​

https://cloud.centos.org/

Ubuntu Server​

https://cloud-images.ubuntu.com/

Debian​

https://cloud.debian.org/images/cloud/

OpenSUSE​

https://download.opensuse.org/repositories/Cloud:/Images:/

OpenEuler​

https://www.openeuler.org/zh/download/

OpenAnonis​

https://openanolis.cn/download

OpenCloudOS​

https://opencloudos.org/ospages/downloadISO

OpenKylin​

https://www.openkylin.top/downloads/index-cn.html

RockyLinux​

https://rockylinux.org/cloud-images/

AlmaLinux​

https://repo.almalinux.org/almalinux/

FreeBSD​

https://download.freebsd.org/ftp/snapshots/VM-IMAGES/
