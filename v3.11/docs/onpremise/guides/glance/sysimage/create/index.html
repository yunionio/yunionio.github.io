еҲ¶дҪңй•ңеғҸ

йҖҡиҝҮ packer еҲ¶дҪңвҖӢ

еҰӮжһңйңҖиҰҒе®ҡеҲ¶еҸ‘иЎҢзүҲзҡ„й•ңеғҸпјҢз”ЁдәҺз»ҷдёҚеҗҢзҡ„дёҡеҠЎдҪҝз”ЁпјҢе°ұйңҖиҰҒиҮӘе·ұеҲ¶дҪңй•ңеғҸгҖӮ

еҸҜд»ҘйҖҡиҝҮдёӢиҪҪеҸ‘иЎҢзүҲж“ҚдҪңзі»з»ҹзҡ„ iso , з„¶еҗҺжң¬ең°еҗҜеҠЁиҷҡжӢҹжңәпјҢе°Ҷ iso е®үиЈ…еҲ°иҷҡжӢҹжңәзҡ„зЈҒзӣҳпјҢз„¶еҗҺдҝқеӯҳиҜҘзЈҒзӣҳпјҢиҝҷдёӘзЈҒзӣҳе°ұеҸҜд»ҘдҪңдёәй•ңеғҸдёҠдј еҲ° glanceпјҢдҪҶжҳҜиҝҷз§Қж–№жі•дәәе·ҘеҸӮдёҺзҡ„жӯҘйӘӨеӨӘеӨҡпјҢе®№жҳ“еҮәй”ҷгҖӮ

жҺЁиҚҗдҪҝз”Ё packer иҝҷдёӘе·Ҙе…·жқҘиҮӘеҠЁеҢ–еҲ¶дҪңй•ңеғҸпјҢиҜҰз»Ҷж“ҚдҪңеҸҜд»ҘеҸӮиҖғеҜ№еә”зҡ„ж–ҮжЎЈ: https://www.packer.io/docs/index.html гҖӮ

https://github.com/yunionio/service-images д»“еә“еҢ…еҗ«дәҶдёҖдәӣжҲ‘д»¬дҪҝз”Ё packer еҲ¶дҪңй•ңеғҸзҡ„й…ҚзҪ®пјҢеҸҜд»ҘеҸӮиҖғдҪҝз”ЁгҖӮ

жүӢеҠЁеҲ¶дҪңжөҒзЁӢвҖӢ

еҜје…Ҙ ISOвҖӢ

  1. жҸҗеүҚеҮҶеӨҮеҘҪж ҮеҮҶISOй•ңеғҸпјҢж”ҜжҢҒз”ЁжҲ·д»Һй•ңеғҸеёӮеңә-ISOз•ҢйқўеҜје…ҘжҲ–зӣҙжҺҘдёҠдј ISOй•ңеғҸгҖӮ
  2. еңЁиҷҡжӢҹжңәеҲ—иЎЁдёӯж–°е»әиҷҡжӢҹжңәпјҢйҖүжӢ©вҖңд»ҺISOеҗҜеҠЁвҖқе№¶йҖүжӢ©еҜ№еә”зҡ„ISOй•ңеғҸпјҢеҲӣе»әжҲҗеҠҹеҗҺпјҢе№¶йҖҡиҝҮVNCз»Ҳз«ҜиҝӣиЎҢжҢүз…§з•ҢйқўжҸҗзӨәе®үиЈ…ж“ҚдҪңзі»з»ҹгҖӮ
жҸҗзӨә
  • жҺЁиҚҗдҪҝз”ЁCentOS Minimalж“ҚдҪңзі»з»ҹгҖӮ
  • Ubuntu/Debianй•ңеғҸеңЁе®үиЈ…иҝҮзЁӢдёӯе»әи®®йҖүжӢ©вҖңNo automatic updatesвҖқе№¶е®үиЈ…OpenSSH ServerиҪҜд»¶гҖӮ
  1. ж №жҚ®й•ңеғҸзҡ„ж“ҚдҪңзі»з»ҹзұ»еһӢиҝӣиЎҢдёҚеҗҢзҡ„дјҳеҢ–й…ҚзҪ®гҖӮ
    • CentOSй•ңеғҸпјҡиҜ·еҸӮиҖғCentOSй•ңеғҸдјҳеҢ–з« иҠӮгҖӮ
    • Ubuntu/Debianй•ңеғҸпјҡиҜ·еҸӮиҖғUbuntu/Debianй•ңеғҸдјҳеҢ–з« иҠӮгҖӮ
    • Windowsй•ңеғҸпјҡиҜ·еҸӮиҖғWindowsй•ңеғҸдјҳеҢ–з« иҠӮгҖӮ
  2. (еҸҜйҖү)еӨҡе№іеҸ°йҖҡз”Ёй•ңеғҸй…ҚзҪ®пјҢеҰӮеҲ¶дҪңзҡ„й•ңеғҸйңҖиҰҒеңЁе…¬жңүдә‘е№іеҸ°дёҠдҪҝз”ЁпјҢйҷӨдёҠиҝ°дјҳеҢ–й…ҚзҪ®еӨ–пјҢиҝҳйңҖиҰҒеңЁLinuxзі»з»ҹе®үиЈ…й…ҚзҪ®cloud-initпјҢWindowsзі»з»ҹе®үиЈ…й…ҚзҪ®Cloudbase-initгҖӮ
  3. й•ңеғҸдјҳеҢ–е®ҢжҲҗеҗҺпјҢйңҖиҰҒе°ҶиҷҡжӢҹжңәе…іжңәгҖӮ
  4. еҚ•еҮ»е…іжңәзҠ¶жҖҒзҡ„иҷҡжӢҹжңәеҸідҫ§ж“ҚдҪңеҲ— "жӣҙеӨҡ" жҢүй’®пјҢйҖүжӢ©дёӢжӢүиҸңеҚ• "дҝқеӯҳй•ңеғҸ" иҸңеҚ•йЎ№пјҢе°ҶиҷҡжӢҹжңәдҝқеӯҳдёәзі»з»ҹй•ңеғҸгҖӮ
  5. й•ңеғҸдҝқеӯҳе®ҢжҲҗеҗҺпјҢз”ЁжҲ·еҸҜеңЁиҷҡжӢҹжңәеҲ—иЎЁдёӯж–°е»әиҷҡжӢҹжңәпјҢйҖүжӢ©вҖңиҮӘе®ҡд№үй•ңеғҸвҖқе№¶йҖүжӢ©дёҠдёҖжӯҘйӘӨдҝқеӯҳзҡ„й•ңеғҸпјҢдҪҝз”ЁеҲ¶дҪңеҘҪзҡ„й•ңеғҸеҲӣе»әиҷҡжӢҹжңәпјҢйӘҢиҜҒй•ңеғҸжҳҜеҗҰеҲ¶дҪңжҲҗеҠҹгҖӮ

CentOSй•ңеғҸдјҳеҢ–вҖӢ

д»ҘCentOS 7 minimalй•ңеғҸдёәдҫӢд»Ӣз»Қй•ңеғҸдјҳеҢ–ж–№жі•гҖӮ

  1. CentOS 7 Minimalж“ҚдҪңзі»з»ҹе®үиЈ…е®ҢжҲҗеҗҺзҡ„иҷҡжӢҹжңәй»ҳи®ӨдёҚиғҪиҒ”зҪ‘пјҢйңҖиҰҒдҝ®ж”№/etc/sysconfig/network-scripts/ifcfg-eth0ж–Үд»¶пјҢе°Ҷ"ONBOOT=no"ж”№дёә"ONBOOT=yes"гҖӮ

    # иҜ·ж №жҚ®е®һйҷ…зҪ‘еҚЎеҗҚз§°дҝ®ж”№еҜ№еә”зҡ„й…ҚзҪ®ж–Үд»¶
    $ vi /etc/sysconfig/network-scripts/ifcfg-eth0    # иҜ·ж №жҚ®е®һйҷ…зҪ‘еҚЎеҗҚз§°дҝ®ж”№еҜ№еә”зҡ„й…ҚзҪ®ж–Үд»¶
    # дҝ®ж”№й…ҚзҪ®ж–Үд»¶еҶ…е®№
    ONBOOT=yes


    # CentOS 8 е’Ң CentOS 9 йҮҚеҗҜзҪ‘еҚЎ: systemctl restart NetworkManager.service

  2. зҰҒз”ЁselinuxпјҢдҝ®ж”№/etc/selinux/configж–Үд»¶пјҢе°Ҷ"SELINUX=enforcing"ж”№дёә"SELINUX=disabled"гҖӮдҝ®ж”№е®ҢжҲҗеҗҺпјҢйҮҚеҗҜзі»з»ҹз”ҹж•ҲгҖӮ

    $ vi /etc/selinux/config
    # дҝ®ж”№й…ҚзҪ®ж–Үд»¶еҶ…е®№пјҢдҝ®ж”№е®ҢжҲҗеҗҺдҝқеӯҳгҖӮ
    SELINUX=disabled

    # йҮҚеҗҜдҪҝй…ҚзҪ®з”ҹж•Ҳ
    $ reboot

  3. е°Ҷеҝ…иҰҒзҡ„kernel moduleеҠ е…ҘеҗҜеҠЁinitram.imgгҖӮпјҲcentos8зҡ„еҶ…ж ёе·Із»Ҹе®үиЈ…дәҶvirtioжүҖд»ҘйңҖиҰҒеңЁеҲ—иЎЁеҲ йҷӨпјү

    $ vi /etc/dracut.conf
    # дҝ®ж”№й…ҚзҪ®ж–Үд»¶пјҢеҺ»жҺүadd_drivers+еүҚйқў#жіЁйҮҠпјҢе№¶еңЁеј•еҸ·дёӯж·»еҠ еҰӮдёӢеҶ…е®№пјҢдҝ®ж”№е®ҢжҲҗеҗҺдҝқеӯҳгҖӮ
    # д»ҘдёӢдёә x86 йңҖиҰҒж·»еҠ зҡ„еҶ…ж ёй©ұеҠЁ
    add_drivers+=" hpsa mptsas mpt2sas mpt3sas megaraid_sas mptspi virtio virtio_ring virtio_pci virtio_scsi virtio_blk vmw_pvscsi nvme "
    # д»ҘдёӢдёә arm йңҖиҰҒж·»еҠ зҡ„еҶ…ж ёй©ұеҠЁ
    add_drivers+=" mptsas mpt2sas mpt3sas megaraid_sas mptspi virtio virtio_ring virtio_pci virtio_scsi virtio_blk nvme "

    # дҪҝй…ҚзҪ®з”ҹж•Ҳ
    $ dracut -f

  4. е…ій—ӯзҪ‘еҚЎжҢҒд№…еҢ–еҠҹиғҪпјҢдҝқиҜҒCentOS 7дёӯзҪ‘еҚЎеҗҚз§°дёәвҖңeth0пјҢeth1вҖқеҪўејҸгҖӮдҝ®ж”№/etc/default/grubж–Үд»¶пјҢеңЁGRUB_CMDLINE_LINUXдёӯж·»еҠ "net.ifnames=0 biosdevname=0"еҸӮж•°гҖӮпјҲcentos8пјҢcentos9з•ҘиҝҮпјү

    $ vi /etc/default/grub
    # дҝ®ж”№й…ҚзҪ®ж–Үд»¶пјҢдҝ®ж”№е®ҢжҲҗеҗҺдҝқеӯҳгҖӮ
    GRUB_CMDLINE_LINUX="crashkernel=auto rd.lvm.lv=centos/root rd.lvm.lv=centos/swap rhgb quiet net.ifnames=0 biosdevname=0"
    # дҪҝй…ҚзҪ®з”ҹж•Ҳ
    $ grub2-mkconfig -o /boot/grub2/grub.cfg

  5. ж №жҚ®йңҖжұӮе®үиЈ…еёёз”ЁиҪҜд»¶гҖӮ

    # д»…дёәдёҫдҫӢпјҢиҜ·ж №жҚ®е®һйҷ…йңҖжұӮе®үиЈ…еёёз”ЁиҪҜд»¶гҖӮ
    $ yum install net-tools git wget vim pcre-tools ntp epel-release -y

  6. зҰҒз”Ёfirewalldе’ҢNetworkManagerжңҚеҠЎгҖӮ

    $ systemctl disable firewalld NetworkManager

  7. еҗҜз”Ёж—¶й—ҙзҪ‘з»ңеҗҢжӯҘпјҢж”ҜжҢҒдҪҝз”ЁntpжҲ–chronyдҝқжҢҒж—¶й—ҙеҗҢжӯҘгҖӮ

    # е®үиЈ…ntpжҲ–chronyиҪҜд»¶
    $ yum install ntp/chrony -y
    # еҗҜз”ЁntpжҲ–chronydжңҚеҠЎ
    $ systemctl enable ntpd/chronyd

  8. дҝ®ж”№ж—¶еҢәдёәCSTгҖӮ

    $ timedatectl set-timezone Asia/Shanghai
    # жҹҘзңӢеҪ“еүҚж—¶еҢә
    $ timedatectl status

  9. sshжңҚеҠЎдјҳеҢ–пјҢдҝ®ж”№/etc/ssh/sshd_configж–Үд»¶пјҢе°ҶPermitRootLoginеұһжҖ§дҝ®ж”№дёәyes е°ҶUseDNSеұһжҖ§дҝ®ж”№дёәnoгҖӮ

    $ vi /etc/ssh/sshd_config

    #еҲҶеҲ«жүҫеҲ°PermitRootLoginеұһжҖ§е’ҢUseDNSеұһжҖ§
    PermitRootLogin yes
    UseDNS no

    systemctl restart sshd

Ubuntu/Debianй•ңеғҸдјҳеҢ–вҖӢ

д»Ҙubuntu-16.04.6-server-amd64.isoдёәдҫӢпјҢд»Ӣз»ҚUbuntuе’ҢDebianй•ңеғҸзҡ„дјҳеҢ–ж–№жі•гҖӮUbuntuзі»з»ҹе®үиЈ…е®ҢжҲҗеҗҺпјҢй»ҳи®ӨдҪҝз”Ёжҷ®йҖҡжқғйҷҗз”ЁжҲ·гҖӮ

  1. sshжңҚеҠЎдјҳеҢ–пјҢдҝ®ж”№/etc/ssh/sshd_configж–Үд»¶пјҢе°ҶPermitRootLoginеұһжҖ§дҝ®ж”№дёәyes е°ҶUseDNSеұһжҖ§дҝ®ж”№дёәnoпјҢиӢҘжІЎжңүдёҠиҝ°еұһжҖ§пјҢиҜ·ж·»еҠ еұһжҖ§гҖӮ

    $ sudo vi /etc/ssh/sshd_config
    # еҲҶеҲ«жүҫеҲ°PermitRootLoginеұһжҖ§е’ҢUseDNSеұһжҖ§
    PermitRootLogin yes
    UseDNS no

  2. еңЁ/etc/init.d/зӣ®еҪ•дёӢеҲӣе»әдёҖдёӘеҗҚз§°дёәssh-initkeyзҡ„иҮӘеҗҜеҠЁи„ҡжң¬гҖӮ

    $ sudo touch /etc/init.d/ssh-initkey
    $ sudo vi /etc/init.d/ssh-initkey
    # и„ҡжң¬еҶ…е®№еҰӮдёӢпјҡ
    #! /bin/sh
    ### BEGIN INIT INFO
    # Provides:          ssh-initkey
    # Required-Start:
    # Required-Stop:
    # X-Start-Before:    ssh
    # Default-Start:     2 3 4 5
    # Default-Stop:
    # Short-Description: Init ssh host keys
    ### END INIT INFO

    PATH=/sbin:/usr/sbin:/bin:/usr/bin
    . /lib/init/vars.sh
    . /lib/lsb/init-functions
    do_start() {
        ls /etc/ssh/ssh_host_* > /dev/null 2>&1
        if [ $? -ne 0 ]; then
            dpkg-reconfigure openssh-server
        fi
    }
    case "$1" in
        start)
        do_start
            ;;
        restart|reload|force-reload)
            echo "Error: argument '$1' not supported" >&2
            exit 3
            ;;
        stop)
            ;;
        *)
            echo "Usage: $0 start|stop" >&2
            exit 3
            ;;
    esac

  3. ssh-initkeyи„ҡжң¬й…ҚзҪ®е®ҢжҲҗеҗҺпјҢиҝҳйңҖиҰҒеўһеҠ еҸҜжү§иЎҢжқғйҷҗпјҢе№¶е°Ҷи„ҡжң¬ж·»еҠ еҲ°зі»з»ҹеҗҜеҠЁи„ҡжң¬зӣ®еҪ•гҖӮ

    $ sudo chmod +x /etc/init.d/ssh-initkey

    Ubuntu 20.04д№ӢеүҚзүҲжң¬пјҢиҜ·жү§иЎҢд»ҘдёӢи„ҡжң¬еҗҜз”Ёи„ҡжң¬пјҡ

    $ sudo /usr/sbin/update-rc.d ssh-initkey defaults
    $ sudo /usr/sbin/update-rc.d ssh-initkey enable

    Ubuntu 20.04зүҲжң¬д»ҘеҸҠд№ӢеҗҺзүҲжң¬пјҢиҜ·жү§иЎҢд»ҘдёӢи„ҡжң¬еҗҜз”Ёи„ҡжң¬пјҡ

    $ sudo /lib/systemd/systemd-sysv-install enable ssh-initkey

  4. пјҲUbuntu 16.04д»ҘдёҠзүҲжң¬и®ҫзҪ®пјүе…ій—ӯзҪ‘еҚЎжҢҒд№…еҢ–еҠҹиғҪпјҢдҝқиҜҒзҪ‘еҚЎеҗҚз§°дёәвҖңeth0пјҢeth1вҖқеҪўејҸгҖӮдҝ®ж”№/etc/default/grubж–Үд»¶пјҢеңЁGRUB_CMDLINE_LINUXдёӯж·»еҠ "net.ifnames=0 biosdevname=0"еҸӮж•°гҖӮ

    $ sudo vi /etc/default/grub
    # й…ҚзҪ®GRUB_CMDLINE_LINUXеҸӮж•°
    GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"
    # дҪҝй…ҚзҪ®з”ҹж•Ҳ
    $ sudo /usr/sbin/update-grub

  5. (иӢҘUbuntuжңӘе…ій—ӯиҮӘеҠЁжӣҙж–°)е…ій—ӯиҮӘеҠЁжӣҙж–°йңҖиҰҒдҝ®ж”№/etc/apt/apt.conf.d/10periodicж–Үд»¶пјҢе°Ҷж–Үд»¶дёӯзҡ„"Update-Package-Lists"еҸӮж•°и®ҫзҪ®дёә0гҖӮ

    $ sudo vi /etc/apt/apt.conf.d/10periodic
    # й…ҚзҪ®дҝ®ж”№
    APT::Periodic::Update-Package-Lists "0";

  6. иҮіжӯӨпјҢиҷҡжӢҹжңәдјҳеҢ–е®ҢжҲҗгҖӮ
